## Quick Start Build Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/NoxNovaViper/3DGame.git
   cd 3DGame
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the project:
   ```bash
   npm run build
   ```
4. Start the project:
   ```bash
   npm start
   ```

Happy Gaming!
